# Quantum APR Projects

This page collects open-source projects and research efforts related to Quantum Automated Program Repair.

## Featured Projects

### Quantum APR Pipeline

Repository:

https://github.com/urja1109/Quantum-APR

Description:

The project encompasses a thorough pipeline to input quantum bugs, generate fixes (CodeT5), filter them, and rank them for best fixes (CodeBERT).
The pipeline refines its fixed code versions through Granite-Qiskit obtained from IBM. The best fixed is therefore selected for each quantum bug.
